# 🧬 BioVitta Laboratórios

O **BioVitta** é um sistema web completo para um laboratório de análises clínicas. O projeto apresenta uma interface moderna para pacientes, incluindo uma área institucional (Sobre, Exames e Unidades) e uma área restrita simulada para consulta de resultados.

---

## 🚀 Funcionalidades

- **Navegação Inteligente**: Menu fixo com scroll suave entre as seções.
- **Área Institucional**:
  - Exibição de Missão, Visão e Valores.
  - Listagem completa de exames disponíveis.
  - Cards interativos de unidades (Curitiba, SP e Rio) com efeito *hover* para visualização de bairros.
- **Área do Paciente (Simulação)**:
  - Sistema de Login via Modal.
  - **Login de Teste**: CPF `123` | Senha `123`.
  - Painel exclusivo com abas: "Meus Exames" e "Dados Cadastrais".
  - Tabela dinâmica de resultados com status.
  - Função de Logout com limpeza de sessão.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as tecnologias fundamentais da web:

- **HTML5**: Estruturação semântica de todo o conteúdo.
- **CSS3**: 
  - Variáveis `:root` para padronização de cores.
  - Flexbox e CSS Grid para layout responsivo.
  - Animações e efeitos de transição nos cards de unidades.
- **JavaScript (Vanilla)**:
  - Lógica de autenticação simulada.
  - Manipulação de DOM para troca de abas e exibição de modais.
  - Controle de fluxo e navegação do usuário.
